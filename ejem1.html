Este código XML genera la siguiente estructura:

  • Pagina inicial (página de nivel superior por defecto)
  • Mi Escuela (con las páginas secundarias Noticias, Anuncios y Trimestre)
  • Clase (con páginas secundarias para cada año (2005, 2004, 2003, 2002 y 2001). Cada una de estas páginas secundarias tiene tres páginas secundarias: Informática, Matemáticas e Inglés)
